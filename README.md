# Everforest Darker for Visual Studio Code

A darker port of the [Everforest](https://github.com/sainnhe/everforest) color
scheme for Visual Studio Code.

## Building and Packaging

This extension targets the latest versions of Visual Studio Code (`^1.96.0`). To create an
installable `.vsix` archive, run the following command:

```sh
npx @vscode/vsce package
```

To install the archive, open the `Extensions` view, click the ellipses, and choose
`Install from VSIX`.
